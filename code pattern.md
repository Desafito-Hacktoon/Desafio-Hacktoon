- Devs de front-end e back-end devem:
  1. Criar uma branch: `feat/001-nome-da-tarefa`
  2. Fazer alterações e `git push`
  3. Abrir Pull Request para `develop`
  4. Aguardar revisão
  5. O DevOps fará o merge

- Ninguém pode fazer `push` direto em `main` ou `develop`
- Pulls da `main` e `develop` são permitidos, mas sem edição
